Winnow Browser Extension

Winnow is a browser extension designed to improve online reading by enabling quick access to Wikipedia. Users can either (1) right-click a selected word or phrase to instantly open its Wikipedia page in a new tab or (2) click the extension icon and search directly from a pop-up window. Unlike Google, Wikipedia offers encyclopedia-style content that’s ideal for building foundational knowledge and exploring interconnected topics.

We brainstormed several ideas for our browser extension including a “Healthy Habits” reminder tool and a Pomodoro timer, but we found that both extensions already existed for the most part. We ultimately focused on right-click search functionality for Wikipedia, aiming for broad appeal and easy use.

Our key design principles were: simple, intuitive interaction (highlight → right-click → search), familiar UI patterns for a minimal cognitive load, universal design for accessibility, and broad applicability(Wikipedia).

We used a basic file structure for our implementation: popup.html and popup.js for the pop-up interface, content.js for right-click functionality, and manifest.json for configuration. We kept the UI minimal, with a clean search bar and visual cues like hover effects. Right-clicking opens the related Wikipedia page in a new tab, and the pop-up allows manual searches.

We interviewed three users who appreciated the extension’s functionality and ease of use. Some common feedback we gathered included: we should add Wikipedia preview text in the pop-up, we should allow repositioning of the icon or persistent icon display, and improve parsing of punctuation and phrasing. Based on this feedback, we enhanced the pop-up’s visual clarity and adjusted the search field to be bigger. In the future, we hope to add multilingual support and preview snippets for better context.
